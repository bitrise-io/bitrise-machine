## Install or upgrade

To install this version, run the following commands (in a bash shell):

```
curl -L https://github.com/bitrise-io/bitrise-machine/releases/download/0.9.0/bitrise-machine-$(uname -s)-$(uname -m) > /usr/local/bin/bitrise-machine
```

Then:

```
chmod +x /usr/local/bin/bitrise-machine
```

That's all, you're ready to call `bitrise-machine`!
